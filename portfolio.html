CASE 01  ·  UX 设计

AI 模拟面试
用户体验优化

2次内面试完成率提升 49%

2024年，AI浪潮下，公司决定开发AI模拟面试模块，我一人成军，完成 UX/UI 以及视觉设计的全部工作，同时借助 AI 工具对面试官进行视觉处理。上线初期遇到面试完成率不高的问题，通过分析与调整，最终成功解决。

01 · 用户调研
深度访谈 + 埋点复盘
访谈12名应届生及跳槽者，结合全量行为埋点，定位真实流失痛点
02 · 数据分析
关键流失节点
23%用户在第1题倒计时结束后跳出——紧张导致未点击"开始作答"
03 · 设计策略
交互 + UI 双线并进
布局重排聚焦视线；沉浸式全屏面试间；AI 面试官形象拟真化
04 · 验证上线
目标超额达成
面试完成率33%→82%，核心指标显著提升，设计目标全面达成
🎙️
沉浸式面试间
16:9 全屏自适应，空间化 UI 复刻真实面试氛围。题干字幕置于屏幕下方，随语音生成自动滚动，沉浸感拉满。
🖱️
交互布局重排
功能按钮高度集中，视线路径最短化，100% 避免找不到"开始作答"。倒计时结束自动录音，降低操作门槛。
🤖
AI 面试官拟真化
自然表情、手势动作、口型吻合，削弱机械感。叠加真实面试环境背景，降低紧张感，减少中途退出率。

CASE 02  ·  UX + 视觉设计

信贷助手
信任破冰

银小智 · AI Credit Assistant

第一次完全独立承接商业项目——甲方在小红书上与我联系，比稿后达成合作，着重对 IP 形象和 UI 进行全链路设计。核心思路是「信任破冰」：分三个关键节点逐步建立用户对信贷产品的信任感，提升借款签约转化率。

📱
破冰第一锤
用户打开 APP 时 以白色 Q 版小老虎「银小智」为 IP 核心，传递干净、安全感。权威标识搭配盾牌图标，突出"持牌机构"，建立第一道信任。
📋
破冰第二锤
用户填写信息时 授权协议中重点突出"用途""有效期"等关键词，缓解隐私焦虑，让用户放心授权并完整填写信息。
✅
破冰第三锤
确认借款时 利率清晰展示并支持自主试算，体现透明无套路。结合 IP 温暖气质，巩固长期信任，提升签约转化率。
🌟
甲方高度评价 · 终获好评

"通过 IP 提升产品信任度的思考甲方也很赞同，通过设计解决产品问题是每位设计师必将经历的课题。合作过程流畅清晰，结果十分满意。"——同时也是我第一次完成完整商业闭环：获客→比稿→设计→协调开发→验收→结款。
